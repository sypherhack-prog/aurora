# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-01-30

### Added
- Initial release of Aurora AI Document Editor
- AI-powered text formatting and generation using Google Gemini
- Rich text editor with Tiptap integration
- User authentication with NextAuth.js
- Subscription system with MVola payment integration
- Admin dashboard for user and subscription management
- Document export in HTML, Markdown, and plain text formats
- Multi-language translation support
- Centralized logging system
- MIT License

### Security
- Password hashing with bcryptjs
- Session-based authentication
- Admin-only route protection
- Selective field fetching to prevent data leaks
